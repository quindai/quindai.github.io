Tutorial - Como instalar o NILMTK no OS X Yosemite(Mac)

O seguinte procedimento foi originalmente adaptado do tutorial oficial para instalação no Ubuntu e outras distribuições Linux.

O tutorial original pode ser encontrado aqui.

É recomendado o uso da Anaconda, que é uma distro free do Python, ela possui mais de 300 pacotes muito usados nas áreas de engenharia, análise de dados e ciências. Recomenda-se instalar a versão Anaconda para Python 2.7.x, segundo os desenvolvedores o NILMTK não suporta a versão 3. Iremos instalar a versão da linha de comandos.


  • Para baixar a versão Python 2.7 clique aqui
    • Após baixar, abra o terminal no seu mac.
    Digite o seguinte no terminal:
    RandyMac:~ randy$ cd ~/Downloads irá navegar até o diretório de Downloads.
    RandyMac:Downloads randy$ bash Anaconda-2.3.0-MacOSX-x86_64.sh

  • Ao finalizar o processo, precisamos atualizar os pacotes da Anaconda, no terminal:
    • RandyMac:Downloads randy$ source ~/.bash_profile
    RandyMac:Downloads randy$ conda update --yes conda, por ora, feche o terminal.
    NOTA: A anaconda já vem com o HDF5 instalado na versão 2.7, verifique aqui os pacotes instalados. No entanto, para garantir iremos instalar algumas dependências extras, para tal iremos usar o MacPorts. Antes de instalá-lo, instale a última versão do Xcode, quando esse artigo foi escrito a versão estável era o Xcode 7.0.1.

    MacPorts é um gerenciador de pacotes que instala pacotes com todas as suas dependências no seu Mac, você pode baixá-lo aqui. Após baixá-lo abra-o usando control+clique e selecione Abrir no menu que aparecer, instale-o.

    Abra um novo terminal e atualize os seus pacotes:
    RandyMac:~ randy$ sudo port selfupdate
    RandyMac:~ randy$ port search hdf5, retorna todos os pacotes relacionados ao formato hdf5.
    Iremos instalar os pacotes 
    lz4, py27-lz4, hdf5, h5utils, h4h5tools
    Usaremos os seguintes comandos no terminal, na mesma ordem que se apresentam de preferência:
    RandyMac:~ randy$ sudo port install lz4
    RandyMac:~ randy$ sudo port install py27-lz4
    RandyMac:~ randy$ sudo port install hdf5
    RandyMac:~ randy$ sudo port install h5utils
    RandyMac:~ randy$ sudo port install h4h5tools

  • Instalando o Pip e outras dependências na distro anaconda, abra um novo terminal:
    • O Mac normalmente já vem com uma versão do Python instalado, para verificar a versão instalada na sua máquina insira o seguinte no terminal:
    RandyMac:~ randy$ python --version
    RandyMac:~ randy$ conda install --yes pip numpy scipy six scikit-learn pandas numexpr pytables dateutil matplotlib networkx, no site oficial do NILMTK essa linha de código inclui o hmmlearn (pacote de Modelos Ocultos de Markov), iremos instalá-lo de forma separada, pois não existe um pacote anaconda para Mac.
